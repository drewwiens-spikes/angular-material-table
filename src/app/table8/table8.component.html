See Also

  • "Virtual Scroll for Angular Material Table"
    • An npm library
    • Has been around a while (~ 2 years)
    • An Angular Directive, which allow to use virtual scrolling in mat-table
    • ~5,000 downloads/week (as of June 2021)
    • Only fixed row size is supported
  • The MatTable virtual-scrolling ticket in the Angular Material issue tracker
    • Created in Feb 2018
    • Still open as of June 2021
  • nGrid, a data table library on top of MatTable
    • From MatTable issue tracker
    • Lots of features
    • Appears to be by one developer, Shlomi Assaf
    • Around ~400-500 downloads/week (as of June 2021)
    • Was in alpha/RC status until ~1 year ago when v2 was published
    • Supports rows with differing height
    • Supports sticky rows or columns inside or outside the scrollable area
  • User garrettld's virtual scrolling MatTable
    • From MatTable issue tracker
    • With sticky rows
    • Assumes each item renders one row = incompatible with multiTemplateDataRows
  • User nahgrin's virtual scrolling MatTable
    • From MatTable issue tracker
    • Relies on a simplified FixedSizeVirtualScrollStrategy = incompatible with AutoSizeVirtualScrollStrategy

Disclaimer: I cannot vouch for the quality of the above links as I have not used them in an app.
